# HELP Site (MVP JSON) — Deploy 100% via Web

## Como publicar SEM usar terminal

1. **No GitHub:**
   - Crie o repositório `helptokenworld/HELP-TOKEN-TESTE-2` (público)
   - Clique em "Add file" → "Upload files"
   - Arraste TODO o conteúdo deste ZIP (pastas e arquivos, não o ZIP inteiro)
   - Clique em "Commit changes"

2. **Na Vercel:**
   - Faça login em https://vercel.com/
   - Clique em "New Project" → "Import GitHub Repository"
   - Selecione `helptokenworld/HELP-TOKEN-TESTE-2`
   - Clique em Deploy
   - Pronto! Seu site estará online em minutos

3. **Atualizações futuras:**
   - Basta subir novos arquivos/pastas no GitHub e commitar
   - A Vercel faz deploy automático

## Estrutura do projeto
- Next.js 14 + Tailwind + i18n (pt/en/es) + tema dark/light
- Dados mock: /data/countries.json e /data/ngos.json
- Páginas: Home, ONGs, Doações, Votações, Transparência, Whitepaper, Como Cadastrar ONG
